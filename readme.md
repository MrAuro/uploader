# Auro Uploader

## Setup:
1. Install dependencies `yarn`
2. Create a `.env` file containing these keys:
```
key=PRIVATE_KEY // This is used for Authorizing your upload requests.
URL=YOUR_DOMAIN
```
3. Start the server. `node index.js`
